# K-Logic Infographic Prompt

## Concepto General
Una infografía que muestra el flujo de transformación de **caos conceptual → orden matemático**, enfatizando la compresión y purificación de información.

## Título Principal
**"K-Logic: Del Ruido al Núcleo Lógico"**
Subtítulo: *Transforma conceptos verbosos en fórmulas precisas*

---

## Elementos Visuales Principales

### 1. **Flujo Central (Izquierda a Derecha)**

**Estilo**: Cascada descendente con flechas gruesas

```
┌─────────────────────┐
│  INPUT ENTROPY      │  (Lado izquierdo: caótico, colorido)
│ (Texto confuso,     │  
│  argumentos largos) │
└──────────┬──────────┘
           ↓ [INPUT_ENTROPY →]
┌─────────────────────┐
│  PHASE 1: IDENTIFY  │  (Extracto variables P₁, P₂, x, y)
│  Variables: x, y    │  [Color: Azul celeste]
│  Predicados: P(x)   │
└──────────┬──────────┘
           ↓
┌─────────────────────┐
│  PHASE 2: PURIFY    │  (Elimina η, ruido)
│  Filtra ruido: η    │  [Color: Azul medio]
│  Guarda lo esencial │
└──────────┬──────────┘
           ↓
┌─────────────────────┐
│  PHASE 3: FORMALIZE │  (Elige dominio)
│  Set theory, Logic, │  [Color: Azul oscuro]
│  Calculus, etc.     │
└──────────┬──────────┘
           ↓
┌─────────────────────┐
│  OUTPUT: LOGIC      │  (Lado derecho: ordenado, matemático)
│  ∀x: P(x) ⟹ Q(x)   │  [Color: Dorado/Blanco sobre fondo oscuro]
└─────────────────────┘
```

### 2. **Dominios Matemáticos (Centro-Derecha)**

Mostrar 5 dominios en "fichas" apiladas/superpuestas:

- 🔢 **Set Theory**: $A ∩ B, x ∈ S, 𝒫(A)$
- 📐 **Predicate Logic**: $∀x, ∃y, P(x) ⟹ Q(y)$
- 📊 **Calculus**: $∫_a^b f(x)dx, ∇f, Δt$
- 🎯 **Linear Algebra**: $\mathbf{F} = m\mathbf{a}, \langle u,v \rangle$
- 🔀 **Category Theory**: $f: A → B, F ⊣ G$

*Cada ficha con un icono distintivo y color gradiente*

### 3. **Métricas de Compresión (Lado Inferior Izquierdo)**

Gráfico de compresión tipo "barras invertidas":

```
Original Length    ████████████████████  100%
After K-Logic      ████                   15%
                   └─ Compression: 85%
```

Lema: *"Máxima formalización = Mínima extensión"*

### 4. **Isomorfismo Check (Lado Inferior Derecho)**

Un símbolo central grande con dos variantes:

- ✅ **≅ Isomorfo** (Sin pérdida)
  - Verde brillante, check mark
  - "Significado = Preservado"

- ⚠️ **≇ No-Isomorfo** (Con pérdida)
  - Rojo suave, bandera
  - "Suplemento mínimo necesario"

## Notas para el Diseñador

- Mantén claridad: incluso sin expertise matemática, el flujo debe ser obvio
- Usa contraste alto (fondo oscuro + colores brillantes)
- Las fórmulas LaTeX deben ser legibles al 100% zoom
- Considera versión "light theme" como alternativa
- Los símbolos matemáticos deben ser profesionales (no "cutifia")
