## 📐 Especificaciones Técnicas Globales

### Resolución
- **Estándar**: 300 DPI (para impresión)
- **Web**: 96 DPI (adjunto en nota de compresión)

## 🎯 Workflow Recomendado para Diseñador

1. **Lee el prompt completo** del skill que vayas a diseñar
2. **Extrae elementos clave**: flujo, colores, tipografía, tamaño
3. **Diseña versión baja-fi** (wireframe/sketch)
4. **Obtén feedback** sobre estructura antes de detalles
5. **Amplifica a alta-fi** con ilustración/vectores profesionales
6. **Testa legibilidad** a diferentes tamaños (móvil, tablet, desktop, impresión)
7. **Genera versión web** (PNG optimizado) + SVG
8. **Crea variantes** (light/dark theme si aplica)

---

## 🔄 Animaciones para Web

Los prompts incluyen animaciones opcionales. Si implementas para web:

- **RECOMENDADO**: 3-5 segundos de duración total
- **Curva de animación**: `ease-in-out`
- **Interactividad**: Hover effects sutiles, no invasivos
- **Performance**: Asegura <16ms frame time (60fps)

Ejemplo: K-Logic puede tener:
```
0s   → Entrada (rojo/caótico) completamente opaca
1.5s → IDENTIFY phase aparece
3s   → PURIFY se revela
4.5s → FORMALIZE se muestra
5.5s → Salida (dorado) brilla
6s   → Loop completo, espera 2s antes de reintentar
```

## 📝 Checklist de Calidad para Cada Infografía

- [ ] Todos los colores usados del código hex especificado
- [ ] Tipografía coincide con recomendación (o equivalente profesional)
- [ ] Elementos matemáticos/símbolos LaTeX son legibles
- [ ] El flujo/proceso es evidente incluso sin leer texto
- [ ] Texto es mínimo, iconografía es máxima
- [ ] Dimensiones están en especificación
- [ ] DPI correcto para formato final
- [ ] Versión light + dark theme (si aplica)
- [ ] Exportado en PNG + SVG + PDF
- [ ] Testeado en móvil, tablet, desktop
- [ ] Callouts integrados (no como post-it)
- [ ] Notas de diseñador fueron consideradas

## 🤝 Cómo Usar Estos Prompts

### Opción A: Con Diseñador Humano
Pasa el archivo `.md` específico al diseñador. Incluye contexto:
```
"Crea una infografía según este prompt.
Es para documentación de una librería de skills de razonamiento.
Audiencia: desarrolladores + product managers.
Tono: profesional, moderno, accesible sin expertise."
```

### Opción B: Con IA (Midjourney, DALL-E, etc.)
Usa el contenido del archivo como base para generar la infografía:
```
"Create an infographic based on this design brief: [copy del prompt relevante]
Style: modern, professional, tech-focused.
Dimensions: [tamaño específico]"
```

### Opción C: Iterativa
1. Genera versión básica (IA o designer)
2. Compara contra checklist de calidad
3. Refina elementos según el prompt
4. Itera hasta 100% alignment con especificación
